Cette prérecette **remplace 0.9.1-prerecette.1** pour la poursuite de la
recette métier. L’historique documentaire et les artefacts d’installation
précédents (`0.9.1-1`) sont conservés ; ils ne doivent pas être écrasés.

## Objet

Correctif d’affichage des anomalies d’import RH
(Lot **2B-IMP-H1** — Affichage exhaustif et défilable des anomalies).

## Cause racine

Troncature UI par `preview.issues.slice(0, 100)`, combinée à l’absence de
défilement vertical dédié. Les erreurs bloquantes pouvaient être masquées
derrière un grand volume d’avertissements.

## Correctifs

- trois filtres : Erreurs bloquantes / Avertissements / Toutes les anomalies ;
- erreurs sélectionnées par défaut lorsqu’elles existent ;
- affichage exhaustif sans plafond silencieux ;
- erreurs placées avant les avertissements dans la vue globale ;
- compteur « affichées sur total » ;
- zone verticalement défilable ;
- résumé d’import clarifié (lignes lues / valides / invalides / erreurs
  bloquantes / avertissements) ;
- affichage du matricule lorsqu’il est disponible.

## Validations automatisées

- `pnpm test` : 618 tests réussis, 0 échec ;
- `pnpm build` : réussi ;
- cas automatisé : 86 erreurs + 480 avertissements (aucune troncature).

## Recette métier encore requise

Avec le fichier Excel réel de recette, contrôler notamment :

- 86 erreurs sur 86 accessibles ;
- 480 avertissements sur 480 accessibles ;
- 566 anomalies sur 566 dans la vue globale ;
- défilement jusqu’à la dernière anomalie ;
- confirmation de l’import toujours bloquée en présence d’erreurs.

La recette métier n’est **pas** validée par cette seule note de livraison.
